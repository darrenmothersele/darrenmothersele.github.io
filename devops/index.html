--- title: DevOps category: devops layout: category splash: Apiary, Puppet, Vagrant... --- My current work in DevOps has been spun off into a separate entity called Apiary. The aim is to be able to deploy websites with your eyes closed, get more development work done (or go home early), and sleep better. Automate infrastructure using Puppet, and deploy to the cloud without having to worry about servers, VMs, and hosts. If you can push to GitHub then you can deploy a website with Apiary. Using the power of Vagrant you can quickly launch a fully working local development environment that emulates your cloud hosting setup. Architect your site as stateless processes and treat backing services as attached resources to enable efficient scale out as demand requires.
