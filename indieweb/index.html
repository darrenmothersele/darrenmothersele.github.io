Am I Indie Yet?

How am I doing on the IndieWeb front?

I'm using IndieMark as a measure of how well I'm doing at implementing IndieWeb.

✓ #ownyouridentity - I've had the darrenmothersele.com domain since early 2007 and I use it for my email. I recently registered daz.is too. This was to make it easy for me to tell people my email address, and so I could create my own short perma-links.

✓ #ownyourdata I'm currently using GitHub pages to host my website for free (static content). I built my own email system, so that's all owned and controlled by me. I've removed myself from most social networks (except Twitter - note to self: sort this out and get POSSEing!). I have data exports where possible so I can backfill my own site.

✓ #useyourownidentity I've got the correct rel=me tags so I can use indieauth. (they fell off for a while, but they're back)

✓ #ownyourdata permalinks and microformats for articles. I want to do some work on permalinks, and give everything a permanent daz.is url. I want to move away from long urls. I don't think they're very human friendly. Didn't people start doing that for SEO? I'm not a fan of SEO. I believe in HBO (human-being optimisation) not search engine optimisation.
